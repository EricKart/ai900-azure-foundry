# Sample Audio

This folder stores audio files generated by the Speech AI notebook.

- `tts_english.wav` - Generated by Text-to-Speech (English)
- `tts_male_voice.wav` - Generated by Text-to-Speech (male voice)
- `tts_french.wav` - Generated by Text-to-Speech (French)

These files are created when you run notebook `04_speech_ai.ipynb`.
They are also used for the Speech-to-Text demonstration.
